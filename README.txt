
-- SUMMARY --

MigrateSourceSqlSrv is an implementation of a MigrateSource for use with
the Migrate module. This source can be used to import from remote Microsoft 
SQL Server database servers. This MigrateSource uses the Sql Server extenstion 
for PHP 5 from Microsoft and is compatible with PHP 5.2 and above.

For a full description visit the project page:
  http://drupal.org/sandbox/Thriuin/1424664
  
Bug reports, feature suggestions and latest developments:
  http://drupal.org/project/issues/1424664


-- REQUIREMENTS --

* Migrate.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

There is no configuration for this module
